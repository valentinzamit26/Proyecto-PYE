\section{Conclusiones}

El estudio permitió caracterizar estadísticamente el tipo de cambio UYU/BRL a partir de una muestra histórica de \TotalRegistros{} observaciones. La variable principal presentó una media de \MediaCambio{}, una mediana de \MedianaCambio{} y un desvío estándar de \DesvioCambio{}, lo que resume su nivel típico y su variabilidad durante el período analizado.

La evidencia gráfica y numérica muestra que la cotización posee estructura temporal, cambios de nivel y posibles valores extremos. Por este motivo, el análisis inferencial debe realizarse con prudencia, especialmente cuando se aplican intervalos y pruebas que suponen independencia o normalidad. Entre las distribuciones candidatas evaluadas, \MejorDistribucion{} fue la de menor estadístico KS para la cotización, aunque este resultado no debe considerarse definitivo sin evaluar el objetivo del modelo y la estructura temporal.

La variable más directamente asociada con la construcción de UYU/BRL es la relación entre las cotizaciones base frente al dólar, especialmente porque la variable principal se calcula a partir de ellas. El análisis de correlación resulta útil para describir asociaciones, pero no habilita conclusiones causales.

Como mejora futura, se recomienda ampliar el análisis con modelos específicos de series temporales, comparar períodos económicos relevantes, estudiar volatilidad condicional y aplicar métodos bootstrap adaptados a dependencia temporal, como bootstrap por bloques. También sería conveniente documentar el código de descarga original para asegurar completa reproducibilidad desde la fuente.
