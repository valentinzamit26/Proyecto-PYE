\section{Pruebas de hipótesis}

El curso presenta la prueba de hipótesis como un procedimiento para evaluar afirmaciones sobre una población a partir de una muestra. Sus etapas principales son identificar un estadístico de prueba, definir \(H_0\), definir \(H_1\), obtener la distribución nula y comparar el valor observado con dicha distribución \cite{curso_hipotesis}. En este informe se utiliza \(\alpha = 0.05\) como nivel de significancia.

\subsection{Normalidad}

Para la cotización UYU/BRL y los log-retornos se planteó:

\[
    H_0: \text{los datos provienen de una distribución Normal}
\]
\[
    H_1: \text{los datos no provienen de una distribución Normal}.
\]

\input{../outputs/tables/normalidad.tex}

Cuando el p-value es menor o igual que \(\alpha\), se rechaza \(H_0\). Si el p-value es mayor, no se rechaza \(H_0\); esto no significa aceptar que la normalidad sea verdadera, sino reconocer que no existe evidencia suficiente para rechazarla con ese criterio. En muestras grandes, incluso diferencias pequeñas pueden producir p-values muy bajos.

\subsection{Correlación entre cotizaciones base}

Se evaluaron correlaciones de Pearson entre variables cambiarias. Para cada par se planteó:

\[
    H_0: \rho = 0
\]
\[
    H_1: \rho \neq 0.
\]

\input{../outputs/tables/correlaciones.tex}

La correlación entre \texttt{brl\_por\_usd} y \texttt{uyu\_por\_usd} fue \CorrBrlUsdUyuUsd{}. Este resultado describe asociación lineal en el período observado, pero no demuestra causalidad. Además, \texttt{uyu\_por\_brl} se construye algebraicamente a partir de ambas cotizaciones base, por lo que no corresponde interpretar esas variables como predictores independientes en sentido causal.

\subsection{Autocorrelación}

Para evaluar dependencia temporal se calcularon autocorrelaciones de los log-retornos:

\[
    H_0: \rho_k = 0
\]
\[
    H_1: \rho_k \neq 0.
\]

\input{../outputs/tables/autocorrelacion_log_retornos.tex}

La autocorrelación permite distinguir entre el comportamiento del nivel de una cotización y el comportamiento de sus retornos. Una autocorrelación alta en niveles no implica necesariamente la misma estructura en retornos. En esta versión no se implementó la prueba Ljung-Box, por lo que queda como mejora posterior para evaluar autocorrelación conjunta hasta varios rezagos.
