\section{Análisis temporal y autocorrelación}

Como la variable se observa en el tiempo, la consigna solicita analizar autocorrelación con diferentes rezagos. Para ello se estudia la relación entre \(X_t\) y valores anteriores de la misma serie, como \(X_{t-1}\) y \(X_{t-2}\). Este análisis no constituye una prueba causal, sino una forma de describir dependencia temporal.
