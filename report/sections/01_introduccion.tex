\section{Introducción}

El tipo de cambio es una variable económica central para interpretar la relación de valor entre dos monedas. En el caso de Uruguay y Brasil, la cotización entre el peso uruguayo y el real brasileño tiene relevancia práctica debido al intercambio comercial, el turismo, las compras transfronterizas y las decisiones financieras que involucran a ambos países.

En este proyecto se estudia la cotización UYU/BRL, definida como la cantidad de pesos uruguayos necesarios para comprar un real brasileño. Dado que se dispone de observaciones diarias a lo largo del tiempo, el fenómeno se aborda como una variable cuantitativa continua observada en una serie temporal financiera. La pregunta principal de investigación es: \textit{¿cómo se comporta estadísticamente el tipo de cambio UYU/BRL en el período analizado y qué modelo probabilístico permite describir razonablemente su distribución empírica?}

El objetivo general es analizar estadísticamente el comportamiento del tipo de cambio entre el peso uruguayo y el real brasileño, representado por la cotización UYU/BRL, a partir de una muestra de cotizaciones históricas, mediante herramientas de estadística descriptiva, análisis exploratorio e inferencia estadística.

Los objetivos específicos son:

\begin{itemize}
    \item describir la variable \texttt{uyu\_por\_brl} mediante medidas de tendencia central, dispersión y posición;
    \item representar gráficamente la evolución temporal, la distribución empírica y posibles valores atípicos;
    \item analizar la autocorrelación de los retornos diarios y la relación entre las series cambiarias base;
    \item proponer distribuciones teóricas candidatas para modelar la variable principal;
    \item construir intervalos de confianza para la media y la varianza;
    \item realizar pruebas de hipótesis pertinentes, considerando las limitaciones propias de una serie temporal.
\end{itemize}

El informe se organiza en ocho secciones. Luego de esta introducción se presenta la metodología de construcción y preparación de los datos. Posteriormente se desarrolla el análisis descriptivo, la evaluación de distribuciones teóricas, la construcción de intervalos de confianza, las pruebas de hipótesis, la discusión de resultados y las conclusiones.
